Titan Endurance Project

Background

The Titan spacecraft system is a two-body system consisting of a launcher and an orbiter capable of deploying spheres into a gravity well that represents the gravitational field of Titan. The launcher launches the orbiter vehicle to pass the gravity well, and the orbiter releases the sphere such that it moves into the gravity well. The sphere represents a landing craft to explore the surface of the moon. The real mission has recently been confirmed by NASA, with an anticipated launch date of July 2028.

My Work

  • Developed a MATLAB script for optimizing trajectories of the lander given the properties of the lander and the target velocities for the round
  • Wrote the firmware for the Arduino
  • Designed the orbiter in SolidWorks
  • Laser cut and 3D printed the parts in the orbiter assembly
  • Specified wheels, servo, chassis wood (material and thickness), optical encoder, battery, and bearings
  • Oversaw project administration including writing reports, updating documentation and logs, tracking project costs, and setting team meetings and deadlines


MATLAB Simulation

I wanted to maximize the time that the sphere spent in the gravity well before dropping out the bottom (for the most points) so I wrote a script for simulation. The script allows the user to input information about the sphere, launch velocity of the orbiter, and launch location relative to the gravity well, and outputs simulation results. The output from the script allows the user to update the Arduino firmware, which controls the launch of the sphere using an optical encoder.
