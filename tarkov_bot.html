Tarkov Market Bot

Background

Escape from Tarkov is a first person shooter with an in game economy and open market trading system. This is a discord bot that can scrape item prices and use the Discord.py library to display information about the tarkov open market in real time. The bot uses these libraries:

  • NumPy
  • BS4
  • Matplotlib
  • Discord.py

Find the project repo here.

Features

  • Implemented current price, price per slot, average 24 hour price, and trader buy/sell, with discord embed
  • Implemented a data trendline of price history with NumPy and Matplotlib
  • Created a custom data repository
  • Created a custom search function
  • Implemented web scraping with BS4 to populate repository
